# ChatGPT Version

This folder contains everything you need to run the Capsule Wardrobe Builder in ChatGPT — either as a Custom GPT (recommended) or as a one-off system prompt in a standard conversation.

## Files in this folder

- **`SETUP.md`** — Step-by-step guide to creating your Custom GPT or using the system prompt in a standard ChatGPT conversation. **Start here.**
- **`SYSTEM_INSTRUCTIONS.md`** — The full system prompt. Paste this into the Custom GPT's "Instructions" field, or into the first message of a standard ChatGPT conversation.
- **`reference_style_tension.md`** — Reference knowledge file for Step 1 (finding your style tension). Upload to your Custom GPT's knowledge base.
- **`reference_palette_example.md`** — Reference knowledge file for Step 4 (colour palette). Upload to your Custom GPT's knowledge base.
- **`reference_formula_template.md`** — Reference knowledge file for Step 5 (outfit formulas). Upload to your Custom GPT's knowledge base.

## Quick start

1. Read [`SETUP.md`](SETUP.md) for full setup instructions
2. Create a Custom GPT (if you have ChatGPT Plus) and paste the system instructions + upload the reference files
3. Start a conversation with "help me build a capsule wardrobe" and follow the 11-step process

## Why ChatGPT specifically

This version takes advantage of ChatGPT's strengths:
- Built-in DALL·E image generation for Step 6 (outfit visualisation)
- Strong image understanding for reading selfies in Step 4 (colour analysis) and wardrobe photos in Step 3
- Custom GPT platform for one-click sharing with others
