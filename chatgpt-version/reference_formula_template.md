# Outfit Formula Template

An outfit formula is not a specific outfit — it's a *pattern* that can be executed in several variations using interchangeable pieces.

## The formula structure

Each formula should specify:

```
Formula [N] — [Short memorable name]
- Layer 1 (top / dress body): [description, colour, fabric]
- Layer 2 (bottom or skirt, if separate): [description, colour, fabric]
- Layer 3 (outer layer if applicable): [description, colour, fabric]
- Legs/feet: [tights/bare, shoe type and colour]
- Accessory: [one distinctive element]

Expresses the tension through: [which tension element each piece carries]

Sub-variations:
- [Swap 1 piece for another that plays the same role]
- [Another swap]
```

## Example — "Joyful maximalism with just enough structure" tension

### Formula 1 — Print Anchor
- Top: cool-toned floral blouse in dusty rose + sage
- Bottom: cool charcoal straight-leg trousers
- Shoes: black ankle boots with small heel
- Accessory: single statement earring

Expresses the tension through: print (loud) anchored by clean charcoal trouser (structure)

Sub-variations:
- Swap floral blouse for plum lace top, same bottom
- Swap charcoal trousers for slim black midi skirt with slit

---

### Formula 2 — Lace and Discipline
- Top: sage lace fitted top
- Bottom: black slim midi skirt
- Legs: sheer black tights
- Shoes: black loafers
- Accessory: small silver stud earrings

Expresses the tension through: lace (soft/loud texture) disciplined by black skirt + tights (structure)

Sub-variations:
- Lace top in dusty rose instead of sage
- Lace dress instead of top + skirt combo

---

### Formula 3 — Body Confident
- Dress: fitted plum midi with slit
- Legs: sheer black tights
- Shoes: black ankle boots with heel
- Accessory: cool berry lipstick, statement earrings

Expresses the tension through: body-con silhouette (confidence) + deep plum (quiet maximalism) + single strong accent (restraint)

Sub-variations:
- Muted burgundy instead of plum
- Add soft black knit cardigan for layering in winter

## Rules for good formulas

1. **Every formula needs an anchor.** One element that brings structure — a sharp silhouette, a grounded colour, a tailored piece. Without the anchor, maximalism becomes chaos.

2. **Every piece appears in 2-3 formulas minimum.** If a piece only works in one outfit, it's not a capsule piece — it's an occasional piece. Either find it more roles or move it out of the core system.

3. **Cover the temperature range.** Include summer versions (lighter fabrics, shorter sleeves, bare legs) and winter versions (layered, tights, boots, knits). Same formula, different weight.

4. **Cover the context range.** If the user has multiple contexts (office + remote, work + weekend, formal + casual), formulas should span all of them. Don't create 10 work outfits for someone who works from home 3 days a week.

5. **Legs/hero features visible.** If the user identified a feature they want to highlight (legs, shoulders, waist, neckline), ensure formulas give that feature room to show. If they want to downplay something, formulas should route attention elsewhere.

6. **The uniform overlay.** If the user has a work uniform that covers part of the outfit (doctor's coat, airline jacket, restaurant apron), design work formulas around what stays *visible*:
   - Neckline
   - Sleeve cuffs
   - Below-coat hemline (knees, legs, shoes)
   These become the expression canvas. The covered torso can be simpler.

## The number game

Aim for 8-12 formulas. Each formula should have 2-4 sub-variations. That gives the user 25-50 distinct outfit combinations from a limited piece count.

The exact piece count depends on how much overlap exists. A wardrobe of 30 pieces that all cross-combine beats a wardrobe of 60 pieces where most combinations don't work.
