# Contributing

Thanks for your interest in improving the Capsule Wardrobe Builder.

## How to contribute

- **Found a bug or awkward interaction?** Open an issue describing what happened and what you expected.
- **Have an idea for a new feature?** Open an issue tagged `enhancement` so we can discuss before you build it.
- **Want to improve the skill directly?** Fork the repo, make your changes, and open a pull request. Please describe what changed and why.

## What kinds of contributions are welcome

- Refinements to the 11-step process based on real usage
- Better prompts or questions in any of the steps
- New integrations (Pinterest analysis, calendar, weather, etc.)
- Translations into other languages
- Typo fixes and clarifications

## What's out of scope

- Adding personal style opinions to the universal builder — it should stay neutral and help users discover their own style tension rather than imposing one
- Tools or integrations that require paid services to function
- Changes that break the skill's portability across platforms (Claude Code, Cowork, OpenAI, etc.)

## Pull request guidelines

- Keep changes focused — one improvement per pull request
- Describe *why* the change matters, not just *what* changed
- If you're adjusting the 11-step flow, explain what user behaviour led you to the change

## Code of conduct

Be kind, be constructive, assume good intent. This is a small project maintained by one person. Response times may vary.
